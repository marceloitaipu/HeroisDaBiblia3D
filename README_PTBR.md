# Heróis da Bíblia 3D — Turbo (Portrait)

Projeto Unity 6 (6000.3.9f1) com 5 mundos:
1) Noé (Runner)
2) Davi x Golias (Boss com pedras + timing)
3) Jonas (Puzzle)
4) Moisés (Puzzle)
5) Jesus (Coleta)

## Como abrir
Unity Hub → Add project → selecione a pasta do projeto.
Abra a cena: `Assets/Scenes/Main.unity` e dê Play.

## GitHub Pages (WebGL)
1) Unity → File → Build Settings → WebGL → Build
2) Faça o Build para a pasta `docs/` na raiz do projeto
3) No GitHub → Settings → Pages → Deploy from branch → `main` / `docs`

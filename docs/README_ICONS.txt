# ⚠️ ÍCONES FALTANDO

Os arquivos `icon-192.png` e `icon-512.png` precisam ser criados.

## Como criar rapidamente:

### Opção 1 - Online (2 minutos)
1. Acesse https://www.canva.com
2. Crie design 512x512px
3. Adicione fundo gradiente roxo/azul
4. Adicione emoji ⚔️ ou texto "HB"
5. Baixe como PNG: `icon-512.png`
6. Redimensione para 192x192px: `icon-192.png`

### Opção 2 - Converter SVG
1. Abra `icon-512.png.svg` no Chrome
2. Clique com direito → Salvar imagem como PNG
3. Renomeie para `icon-512.png`
4. Delete os arquivos .svg depois

### Opção 3 - Placeholder rápido
```powershell
# Se quiser usar imagem qualquer temporariamente
# Coloque qualquer PNG 512x512 e renomeie para icon-512.png
# Coloque qualquer PNG 192x192 e renomeie para icon-192.png
```

**Enquanto não tiver ícones:** O PWA funcionará, mas sem ícone bonito na home screen.

Veja detalhes em: `ICONS.md`
